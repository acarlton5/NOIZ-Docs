Finding Games & Adding Custom Game Tags

How Games Work on NOIZ

Games are at the core of stream discovery, Quest routing, and viewer engagement on NOIZ. Each stream can be assigned to a recognized game or category, helping the platform understand the content’s context and route Quests, boosts, and Echostage placement accordingly.

Categories aren't limited to just games — streamers can also select broader labels like “Just Talking,” “Creative,” or “Music” to better represent non-gameplay content. Each of these categories may feature platform-curated or sponsored Quests and appear in discovery tools across NOIZ.

Finding the Right Game Tag

When setting up your stream, the Game Tag selector helps you find the right match for your content. Simply type the name of the game or category, and a list of available matches will appear.

Each game or category tag includes metadata such as:

  • Genre (e.g., Action, Puzzle, Roguelike)
  • Platform (PC, Console, Mobile)
  • Developer or Publisher (if verified)

Games linked to official publisher accounts on NOIZ are marked with a verified badge, making them eligible for sponsored Quests or official directory boosts.

Requesting a New Game Tag

If the game you're streaming doesn’t appear in the list, you can submit a request through the Suggest a Game tool, located in your stream setup panel. This tool lets you provide all necessary details:

  • Game title
  • Developer or publisher name
  • Official website or storefront link
  • Cover art or promotional image

Once submitted, the game will go through internal review before being added to the public directory. You’ll be notified once your suggestion is accepted or if more info is needed.

Using Custom Stream Tags

Beyond game or category selection, custom tags allow creators to further define the tone or content of a stream. These tags might describe playstyle, objectives, or community focus. Common examples include:

  • Speedrun
  • First Playthrough
  • Competitive
  • Modded
  • Co-op
  • Chill Vibes

Custom tags affect:

  • Viewer discovery filters
  • Echostage routing decisions
  • Quest assignment relevance (e.g., “watch a modded run”)

Use tags to guide the type of viewers and interactions you want to attract.

Creating Channel-Specific Tags

Creators with Partner or Partner+ status can create custom tag presets for their own channel. These allow for flexible re-use and automation:

  • Set default tags for each stream category
  • Trigger tags dynamically via bots or plugins
  • Let viewers unlock temporary tags through quests or Reverb rewards

These custom tags are cosmetic but help reinforce a channel's brand and discoverability.

Tag Visibility and Discovery

All active tags are displayed:

  • On your stream thumbnail
  • Inside your channel description
  • On game and category pages

NOIZ’s systems use tag metadata to:

  • Refine search results and filters
  • Recommend your stream to relevant viewers
  • Route platform Quests and highlight eligible content for Echostage

Tag accuracy improves viewer experience and helps the platform better promote quality streams.

Best Practices for Tagging

  • Be accurate and honest. Misleading tags reduce your visibility and may flag your stream for moderation.
  • Don’t over-tag. Stick to 3–5 meaningful tags that actually describe your content.
  • Update tags when stream content shifts. If you switch games or change tone midstream, your tags should reflect that.
  • Use tags as an extension of your stream’s identity. Let them help shape the viewer experience from first glance.

FAQ

How long does it take for a suggested game to be reviewed?
Most requests are reviewed within 3–5 business days, but sponsored or trending games may be prioritized.
Can I tag multiple games in one stream?
Only one game can be selected per stream, but you may describe others in your tags (e.g., “Variety Night”).
Are tags required?
Game or category tags are required. Custom tags are optional but strongly recommended.
Do tags affect monetization or quests?
Yes — some Quests are tag-sensitive (e.g., “Complete a Co-op stream”), and proper tagging can boost visibility and Echostage routing. Monetization eligibility itself is not impacted.
