NOIZ Studio is the platform’s official streaming app, designed to make high-performance, low-latency streaming simple — whether you're brand new or a seasoned broadcaster. It’s built on an optimized fork of OBS and includes native ITG relay support, chat tools, and integrated Quest features.

1. What Is NOIZ Studio?

NOIZ Studio is a modified version of OBS (Open Broadcaster Software), enhanced specifically for the NOIZ ecosystem. It’s designed to:

  • Connect directly to NOIZ’s ultra-low latency ITG relay network
  • Automatically optimize video and encoding settings
  • Seamlessly integrate with chat, Quests, overlays, and support tools

Unlike standard OBS, NOIZ Studio:

  • Detects your region and selects the best relay node
  • Preconfigures your encoder and bitrate settings for ITG
  • Includes built-in chat, Quest controls, and stream previews

2. Getting Started

2.1 Download & Launch

  • Download NOIZ Studio from your Streamer Dashboard or stream.noiz.gg
  • Install and launch the app (auto-updates are built in)

2.2 Link Your Constellation Session

  • A 6-digit token will appear on launch
  • Open the Constellation app → tap Active Connections → enter the code to link your account

2.3 Run First-Time Setup

NOIZ Studio’s Setup Wizard will guide you through:

  • Microphone & camera selection
  • Bandwidth & encoder test
  • Default stream category selection
  • Choosing a layout template

3. Stream Configuration

Once linked, Studio auto-configures the following:

  • Ingest Server: Closest available ITG relay node
  • Bitrate: Based on your test results
  • Encoder: Prefers hardware (NVENC, AMD, etc.) if available
  • Stream Delay: Set to 0ms unless you adjust it manually

You can change any setting by going to:

  • Settings → Output
  • Settings → Stream

4. Scenes & Layouts

NOIZ Studio supports full OBS-level customization with added features:

  • Scene collections (saved locally and to the cloud)
  • Source groups for easier overlay management
  • Plugin-ready scene elements (chat box, emotes, Quest tracker, etc.)

Pro Tip: Browse the built-in NOIZ Plugin Hub to install interactive overlays, theme packs, and verified widgets directly from inside the app.

5. Relay Routing

By default, NOIZ Studio:

  • Automatically routes your stream to the best-performing relay node
  • Displays your active relay status in the HUD panel
  • Fails over silently in the event of node instability

You can override the selected relay by going to:

  • Settings → Advanced → Relay Selection

If you’re a Pathweaver host, Studio will prioritize your relay node when possible.

6. Monitoring Your Stream

The live performance panel (top-right of Studio) shows:

  • Bitrate stability
  • Frame drops
  • Relay ping (ms)
  • Chat connection health
  • Plugin status

⚠️ If connection issues occur:

  • NOIZ Studio will attempt automatic failover to the next-best relay
  • Viewers will experience no disconnects; you’ll see a HUD notification

7. Stream Controls & Features

Once linked to NOIZ, you’ll see new UI options:

  • Start / Stop Stream button (platform-linked)
  • Stream metadata panel: title, category, tags
  • Viewer count and ⚡︎dB support alerts
  • Hotkey support for future Clip creation and Quest control

You can still create multiple scene collections, add browser sources, and use OBS plugins as usual.

8. Frequently Asked Questions

Can I use NOIZ Studio to stream to other platforms?

Dual streaming is supported via custom plugins, but NOIZ Studio is optimized for NOIZ-first performance.

Can I import my old OBS scenes or profiles?

Yes — on first launch, you’ll be prompted to import any existing OBS configurations.

Do I need to reinstall my plugins or VSTs?

Only if you use custom audio plugins or complex source filters. NOIZ-native plugins are included.

Is it safe to run a Pathweaver relay on the same machine?

Yes — NOIZ Studio will detect it and auto-route. Port and conflict management is handled internally.
