2. What Determines Action

The DCT reviewer evaluates:

  • Was there actual harm, threat, or disruption?
  • Was the stream or content correctly tagged?
  • Has the user been warned before?
  • Was the behavior clearly malicious, exploitative, or manipulative?

DCT uses a hybrid approach:

  • Rule triggers (e.g., slurs, hate symbols, flagged URLs)
  • Pattern recognition (e.g., known troll groups, repeat usernames)
  • Manual judgment (e.g., tone, severity, context of violation)

6. Reversal & Appeal

If a user appeals successfully:

  • Action is reversed, reduced, or annotated
  • A note is added to the user's moderation history

If the appeal is denied:

  • A system message explains the decision
  • Escalation to Staff Admin is noted (if applicable)

7. DCT Bias Protection

To prevent misuse or favoritism:

  • DCT reviewers cannot moderate:
    • Friends or followers
    • Creators they support
    • Known rivals
  • Automatic escalation occurs if:
