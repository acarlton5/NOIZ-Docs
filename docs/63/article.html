Category Tags and Stream Metadata Best Practices

For streamers using titles, categories, tags, and quest metadata

Accurate tagging helps viewers find your content, improves platform safety, and ensures quests and campaigns register correctly. Mismatched or misleading metadata may reduce visibility or lead to moderation.

Why Tags and Metadata Matter

Tagging and stream metadata on NOIZ are used for:

  • 📡 Search & discovery
  • 🎯 Quest progress tracking
  • 🧠 Maturity & content routing
  • 🌍 Relay node optimization

Incorrect metadata can cause:

  • Viewer confusion or drop-off
  • Quest progress being denied
  • ⚡︎dB disputes or monetization freezes
  • Trust penalties in visibility algorithms

How Categories Work on NOIZ

Each stream must include a Primary Category, chosen from:

  • 🎮 A specific game (via the Game Directory)
  • 🧑‍🎤 A general stream type (e.g. Just Chatting, IRL, Music, Art)
  • 🛠️ A development or creative mode (e.g. Game Dev, 3D Modeling)

✅ Streamers may also:

  • Add custom games or titles (reviewed for approval)
  • Propose new categories via “Submit Game / Category” in the dashboard

Custom titles display locally until verified.

Using Tags Effectively

Tags help viewers understand what your stream is about. Common tag types:

  • 🎮 Genre: FPS, Horror, Chill, Puzzle
  • 🎭 Style: Solo Play, Co-op, Speedrun, Roleplay
  • 🔞 Rating: 18+, Mature, Family-Friendly
  • 🧠 Identity: LGBTQ+, VTuber, Accessibility Enabled

💡 Best practices:

  • Use 3–6 tags — quality over quantity
  • Do not tag “Giveaway,” “Partner,” or “Drops” unless relevant
  • Update or remove expired tags between sessions

Metadata That Impacts Discovery

Your full stream metadata includes:

  • Stream title
  • Primary category
  • Tags
  • Language setting
  • Region (auto-detected for relay routing)
  • Visibility: Public / Unlisted / Relay-only

Viewers filter by:

  • Category
  • Tag
  • Language
  • Stream features (e.g. “Sub-Only”, “Plugin Enabled”, “Quest Stream”)

Quest-Specific Tagging Rules

When participating in Quests:

✅ You must:

  • Match your category and tag to the Quest requirement
  • Use correct event hashtags (e.g. #E3Showcase, #IndieWeek)

❌ You may not:

  • Claim Quest participation without actual involvement
  • Leave required tags off and expect progress to track

Failure to follow Quest tagging rules may result in:

  • ⚡︎dB withholding
  • Quest goal rollback
  • Campaign disqualification

Metadata Moderation

The DCT may flag or override metadata if:

  • Tags are inaccurate or misleading
  • Tags are used to bypass maturity filters
  • The category contradicts the stream’s actual content

Consequences of tag abuse:

  • ⚠️ Tagging cooldowns
  • 🛑 Quest bans
  • 📉 Lowered visibility in browsing & recommendations

Tips for Getting Found

✅ Improve discoverability:

  • Tag your style (e.g. No Commentary, Developer Q&A)
  • Use consistent title formatting across streams
  • Include game versions or special modes (e.g. “1.20 Hardcore”, “Demo Build”)
  • Update tags mid-stream when content changes

FAQ

Can I create my own tags?
Yes. Custom tags appear locally and may be verified by staff later.
Do tag orders affect my search ranking?
No. Relevance matters more than order.
Can I use tags from Twitch or YouTube?
Yes, if they are meaningful and context-appropriate.
How long does it take for a new game or category to be approved?
1–3 business days, depending on queue volume.
