Overlays and tools on NOIZ add visual flair and interactivity to your stream and can be installed or managed directly in NOIZ Studio or OBS.

1. What Are Overlays and Stream Tools?

On NOIZ, overlays and tools are modular assets that enhance the visual, informational, or interactive experience of a stream. These include:

  • Camera borders, stingers, alert boxes
  • Plugin-driven tools (timers, hype trackers, emote bursts)
  • Custom scenes or transition packs
  • Marketplace-installed assets or personal uploads

They can be:

  • Imported directly into NOIZ Studio or OBS
  • Managed through the Plugin Manager or Scene Layout Editor

2. How to Install Overlays from the Marketplace

  1. Visit the Marketplace → Overlay Packs
  2. Choose a pack or tool labeled:
    • NOIZ Studio Compatible
    • OBS Import Bundle
  3. Click "Add to My Assets" or "Install Now"
  4. Open NOIZ Studio → Assets to find your new overlays
  5. Drag-and-drop into your active scene or scene list

Overlays may also come with .overlay, .json, or .npk config files for faster setup.

3. Managing Overlays in NOIZ Studio

In NOIZ Studio:

  • Use the Layout Panel to add/remove visual elements
  • Layer items by drag-and-drop
  • Lock positioning for screen-safe zones
  • Toggle overlays per scene or profile
  • Preview mobile vs desktop stream layout

You can also:

  • Bind overlays to plugin events (e.g. emote rain after sub)
  • Adjust opacity, size, and animation timing
  • Group overlays into a reusable Scene Template

4. Custom Overlay Uploads

✅ You can upload your own assets:

  • Static PNG/JPEG overlays
  • Transparent WebM animations
  • Lottie files for lightweight vector effects
  • Custom plugin graphics or stream borders

To upload:
Dashboard → Creator Tools → Upload Overlay

You can categorize by:

  • Scene type (BRB, Live, Starting Soon)
  • Feature (Alert, Frame, Tracker)
  • Source (Self-made, Commissioned, Purchased)

5. Linking Overlays with Plugins

Some overlays are plugin-reactive:

  • Change appearance when a sub or ⚡︎dB tip occurs
  • Visually track Quest completion
  • Trigger when a Hype Moment is detected
  • Allow viewer reactions to affect visual effects

These can be configured under:
Dashboard → Plugins → Overlay Triggers

6. Performance & Compatibility

Overlays are rendered client-side for viewers. Tips to optimize:

  • Limit simultaneous WebM animations to <3 on low-end systems
  • Avoid 4K overlays unless your stream targets high-spec playback
  • Use overlay presets tested with NOIZ Studio or OBS 29+
  • Keep total overlay file size under 25MB per scene for mobile compatibility

7. FAQ

Can I sell overlays on the Marketplace?
→ Yes, if you’re a Verified Artist

Can I import Twitch or Streamlabs overlays?
→ Not directly, but you can repackage the assets

Can overlays use audio?
→ Yes — with autoplay disabled and viewer opt-in

Do overlays affect latency?
→ No — they’re optimized and rendered separately from your video feed
